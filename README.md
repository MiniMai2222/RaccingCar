HƯỚNG DẪN CÁCH CHẠY GAME:

**Lưu ý:**

* Game được làm dựa trên nền tảng VS CODE, yêu cầu cài trước kèm mingw64 hoặc msys2, làm theo:

https://code.visualstudio.com/docs/cpp/config-mingw

* Link thư viện SDL3:

https://github.com/libsdl-org/SDL/releases

* Link thư viện SDLttf:

https://github.com/libsdl-org/SDL\_ttf/releases

* Trong file gửi đã có file **Makefile**, copy rồi dán vào terminal, Enter, sau khi chương trình biên dịch xong sẽ xuất hiện 1 file đuôi exe.
* Chạy game bằng cách mở file exe hoặc ghi **.\\main.exe** vào terminal.
